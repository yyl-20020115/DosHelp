Returns a string representation of a variable's address for use in DRAW and
PLAY statements.

VARPTR$(commandstring$)

    ■ commandstring$    A string variable containing DRAW or PLAY commands.

Example:
    Scale$ = "CDEFGAB"
    PLAY "L16"
    FOR i% = 0 TO 6
        PLAY "O" + STR$(i%)
        PLAY "X" + VARPTR$(Scale$)
    NEXT i%
