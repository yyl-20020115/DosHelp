Renames a file or directory.

NAME oldspec$ AS newspec$

    ■ oldspec$ and newspec$    The name of an existing file and the new name
                               for the file. Each name may include a path.

Example:
    INPUT "Old Name: "; OldFN$
    INPUT "New Name: "; NewFN$
    NAME OldFN$ AS NewFN$
